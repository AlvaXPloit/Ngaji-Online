## ✨ Fitur

- 🌙 Dark mode (opsional)

---

## ⚙️ Cara Instalasi

1. Clone repository:
   ```bash
   git clone https://github.com/AlvaXPloit/ngaji-online.git
   cd ngaji-online
   ```
2. Jalankan server lokal:
   ```bash
   php -S localhost:8000
   ```
3. Buka browser ke:  
   `http://localhost:8000`
